OPTICS Clustering — Detailed Explanation

OPTICS (Ordering Points To Identify the Clustering Structure) is a density-based clustering algorithm used to identify natural groupings in data without requiring the number of clusters to be predefined. It is an advanced version of DBSCAN and works well for complex datasets.

The algorithm groups points based on density. Points that are close together form clusters, while distant points are considered anomalies. This makes OPTICS powerful for detecting patterns in real-world data.

In this project, the dataset contains (x, y) coordinates. Each row represents a point. The algorithm calculates distances and groups nearby points.

Example: Points like (1,2), (2,3), (3,2) form one cluster. Points like (20,25), (21,26) form another cluster. Points like (500,500) are anomalies.

Unlike K-Means, OPTICS does not require specifying the number of clusters. It automatically discovers clusters based on density.

The results are visualized using an interactive graph where clusters are colored differently and anomalies are highlighted.

How This Project Was Developed

This project was created to demonstrate OPTICS clustering using a web interface. It combines machine learning with visualization.

The system is divided into frontend and backend. The frontend handles UI, and backend performs clustering.

Frontend uses HTML, CSS, JavaScript for design and interaction.

Backend uses Python Flask and OPTICS algorithm from scikit-learn.

Plotly.js is used for interactive graphs with zoom and selection tools.

The system was tested with multiple datasets for accuracy and performance.

How to Use This Website

Step 1 — Upload dataset (CSV with x,y columns).

Step 2 — Enter min_samples value.

Step 3 — Click Run Clustering.

Step 4 — View clusters in graph.

Step 5 — Identify anomalies visually.

Step 6 — Download results.
